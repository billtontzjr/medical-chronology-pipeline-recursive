# Quality Checker

Perform the "Phase 3: Self-Correction & Refinement Loop" from ../CLAUDE.md.

Review the generated chronology.md file and check:

1. **Formatting:**
   - Every entry follows: Date. Facility. Provider. Visit Type.
   - Proper header format
   - NO bold, bullets, or citations

2. **Tone:**
   - Direct, factual (not "patient was seen for...")
   - In-paragraph headings used correctly

3. **Content:**
   - Imaging reports: impression only
   - Physical exams: orthopedic/spine focus
   - Therapy: properly consolidated

4. **Verification:**
   - Every entry from actual document
   - Dates accurate
   - Medical terms correct

Report any issues found and suggest corrections.
